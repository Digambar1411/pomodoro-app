# Pomodoro Web App

## Quick Start

Preview Link : https://neo-focus.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/Digambar1411/pomodoro-app.git
```

## Features Available.

- Home Page
- settings
- Task page
- Timer page
- Authentication
  - Login
  - Sigup
  - Guest Login
- Resposive Web app

## Tech Stack

**Client:** React, Context API, React Router v6.

**Server:** Mockbee.
